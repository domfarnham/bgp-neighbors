BGP Neighbors, By Dom Farnham

Another Microservice By Dom Farnham

A microservice which provides support information quickly.

Provides network operations engineers with the intial information they need in order to respond to a BGP Neighbor status change.

Example usage:

https://bgpneighbors.glitch.me/neighbor/1.1.1.1

Example output:

{ "hostname": "router1", "ip": "1.1.1.1", "asn": "64888", "cid": "xxxx-xxxx-xxxx", "supportDoc": "https://example.com/router1.pdf" }
